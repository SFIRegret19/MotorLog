# MotorLog Test Cases

Дата обновления: 12.04.2026
Версия покрытия: обновленное приложение с weather-виджетом, AI-чатом, заправками и аналитикой

## Функциональные тест-кейсы

| ID | Приоритет | Область | Предусловия | Шаги | Ожидаемый результат |
|---|---|---|---|---|---|
| TC-001 | P0 | Запуск приложения | Приложение установлено | 1. Открыть приложение | Открывается экран `MotorLog` (гараж), без падений |
| TC-002 | P0 | Добавление авто (валидация) | Открыт экран добавления авто | 1. Нажать `Сохранить` с пустыми полями | Показан `SnackBar` с требованием заполнить поля |
| TC-003 | P0 | Добавление авто (успех) | Гараж пуст | 1. Выбрать марку и модель 2. Ввести год и пробег 3. Сохранить | Авто появляется в гараже, для него созданы дефолтные расходники |
| TC-004 | P0 | Удаление авто | В гараже есть минимум 1 авто с расходниками/заправками | 1. Нажать удаление 2. Подтвердить | Авто исчезает из списка; связанные записи не доступны в интерфейсе |
| TC-005 | P0 | Обновление пробега (валидный сценарий) | Есть авто с расходниками | 1. Открыть обновление пробега 2. Ввести большее значение 3. Сохранить | Пробег обновлен; износ расходников увеличился; показан успех |
| TC-006 | P0 | Обновление пробега (некорректный сценарий) | Есть авто | 1. Ввести пробег меньше текущего 2. Сохранить | Данные не меняются; показана ошибка валидации |
| TC-007 | P0 | Экран расходников | У авто есть расходники | 1. Открыть `ТО` | Отображается список деталей и текущий износ |
| TC-008 | P1 | Добавление расходника из справочника | Открыт экран расходников | 1. Нажать `+` 2. Выбрать деталь из списка 3. Сохранить | Новая деталь добавлена в список |
| TC-009 | P1 | Добавление кастомного расходника | Открыт экран добавления расходника | 1. Выбрать `Свой вариант...` 2. Ввести имя и ресурс 3. Сохранить | Кастомная деталь добавлена с износом 0% |
| TC-010 | P0 | Замена расходника | Есть деталь с износом > 0 | 1. Нажать `refresh` у детали 2. Ввести стоимость 3. Подтвердить | Износ детали сброшен до 0%; добавлено сервисное событие |
| TC-011 | P1 | Удаление расходника | В списке есть деталь | 1. Нажать удаление 2. Подтвердить | Деталь удалена из списка |
| TC-012 | P0 | Добавление заправки (валидация пустых полей) | Открыт экран новой заправки | 1. Оставить поля пустыми 2. Нажать `Сохранить` | Показан `SnackBar` о необходимости заполнить поля |
| TC-013 | P0 | Добавление заправки (валидация пробега) | Текущий пробег известен | 1. Ввести пробег меньше текущего 2. Сохранить | Запись не создается; показана ошибка про пробег |
| TC-014 | P0 | Добавление заправки (успех) | Есть авто | 1. Ввести корректные данные 2. Сохранить | Создана запись о заправке; при большем одометре обновлен пробег авто |
| TC-015 | P0 | Аналитика расходов | Есть заправки/сервисные события | 1. Открыть `Статистика` | Отображаются общие затраты и суммы по категориям |
| TC-016 | P1 | Детализация топлива | Есть минимум 1 заправка | 1. В аналитике открыть `Топливо` | Показан список заправок с литрами, суммой, датой и пробегом |
| TC-017 | P1 | Детализация сервиса | Есть минимум 1 сервисное событие | 1. В аналитике открыть `Сервис и ТО` | Показан список сервисных работ с датой и стоимостью |
| TC-018 | P1 | Калькулятор поездки | Открыт экран аналитики | 1. Перейти в калькулятор 2. Ввести расстояние и цену | Корректно пересчитываются литры и стоимость поездки |
| TC-019 | P0 | Weather-виджет: успешная загрузка | Включены интернет и геолокация, валидный API-ключ | 1. Открыть гараж | Показаны описание погоды, температура и совет по эксплуатации |
| TC-020 | P0 | Weather-виджет: ошибка/нет доступа | Отключен GPS или нет разрешения | 1. Открыть гараж | Показан блок `Погода недоступна` с кнопкой обновления |
| TC-021 | P1 | Weather-виджет: ручное обновление | Weather-виджет видим | 1. Нажать `refresh` | Появляется индикатор загрузки и затем новое состояние погоды |
| TC-022 | P1 | AI-чат: стартовое состояние | Есть карточка авто | 1. Открыть `AI-Совет` | Показано приветствие ассистента с информацией о текущем авто |
| TC-023 | P1 | AI-чат: ответ по ключевым словам | Открыт AI-чат | 1. Отправить вопрос про масло/ТО/погоду | Ассистент возвращает релевантный сценарный ответ |
| TC-024 | P2 | AI-чат: fallback-ответ | Открыт AI-чат | 1. Отправить произвольный вопрос вне сценариев | Возвращается общий fallback-ответ |

## Нефункциональные проверки

| ID | Приоритет | Проверка | Шаги | Ожидаемый результат |
|---|---|---|---|---|
| NTC-001 | P0 | Стабильность навигации | Последовательно пройти: гараж → ТО → статистика → калькулятор → назад | Нет крэшей, корректные возвраты на экраны |
| NTC-002 | P1 | Стабильность после поворота экрана | На экранах с формами повернуть устройство | Данные формы не теряются в пределах текущей сессии экрана |
| NTC-003 | P1 | Корректность локализации UI | Просмотреть ключевые экраны | Тексты на русском, без обрезаний и артефактов |
